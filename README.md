# greencity-tests

Даний репозиторій містить навчальну тестову документацію та автоматизовані тести для порталу GreenCity. Робота виконана в межах вивчення основ тестування ПЗ та автоматизації згідно зі стандартами SoftServe IT Academy.

## Опис проєкту
Метою роботи було проведення аналізу сторінки подій порталу GreenCity, створення набору тест-кейсів та їх подальша автоматизація. Проєкт включає:
* Аналіз інтерфейсу та функціональних можливостей сторінки.
* Написання позитивних та негативних тест-кейсів.
* Оформлення документації з використанням мови розмітки Markdown.
* Реалізацію автоматизованих сценаріїв на мові Python.

## Посилання на проєкт
* Сайт, що тестується: https://www.greencity.cx.ua/#/greenCity/events

## Об'єкти тестування
Здійснено перевірки для наступних функцій:
1. Фільтрація: Перевірка сортування подій за типом (позитивні та негативні сценарії).
2. Вибір категорій: Перевірка функціонала кнопки вибору всіх типів подій.
3. Пошук: Перевірка роботи пошукового поля.

## Технічні особливості реалізації
Автоматизація виконана за допомогою бібліотеки Selenium WebDriver та стандартного модуля unittest. 
* Реалізовано 3 основні позитивні тести та 3 відповідні негативні сценарії.
* Тест-кейс ТС-4 (негативний пошук) реалізовано з використанням параметризації через метод subTest, що дозволяє перевіряти декілька наборів даних в межах одного тесту.

## Інструкція запуску тестів
Для запуску всіх автоматизованих тестів необхідно використати наступну команду в терміналі:

python -m unittest discover tests
