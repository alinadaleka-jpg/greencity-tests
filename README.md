# greencity-tests
Даний репозиторій містить навчальну тестову документацію для порталу **GreenCity**. Робота виконана в межах вивчення основ тестування ПЗ  згідно зі стандартами SoftServe IT Academy.
## Опис завдання
Метою роботи було проведення аналізу та створення набору тест-кейсів. 
Основні завдання включали:
* Аналіз інтерфейсу та функціональних можливостей сторінки.
* Написання позитивних та негативних тест-кейсиів.
* Оформлення документації з використанням мови розмітки **Markdown** в екосистемі GitHub.

## Посилання на проєкт
* **Сайт, що тестується:** [GreenCity Events](https://www.greencity.cx.ua/#/greenCity/events).

## Об'єкти тестування
У документації описано перевірки для наступних функцій:
1.  **Фільтрація:** Перевірка сортування подій за типом.
2.  **Авторизація:** Процес входу в систему з використанням  невалідних даних.
